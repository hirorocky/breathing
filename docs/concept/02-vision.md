# 場のビジョン

## 核

存在しやすい空間を探求するための Web サイトである。

「存在しやすい」とは何か、どんな要素が必要か——答えを先に決めず、動く場を置きながら探る。サイトを修正し、観察し、得た気づきを次のバージョンへ渡す。このループを回し続けることが、プロジェクトの中心にある。

多くの場所では速さ・強い主張・即時反応・社交性・情報処理・強い熱量が暗黙に求められる。ここで試しているのは、呼吸・余白・間・静けさ・非攻撃性・曖昧さ・無理のない接続が、存在負荷を下げるかどうかである。

人を導く空間ではない。無理に反応しなくていい。何者かにならなくていい。静かなままで存在できる——そう感じられる空気の断片を、一つずつ実在させながら探求する。

## 空気の質感

静か、呼吸感がある、少し孤独だが閉じていない、余白がある、説明しすぎない、思索的、夜っぽい、やわらかい、寡黙、温度は低めだが冷酷ではない、非攻撃的、未完成を許す、少し不器用、かわいいが幼くない、かっこいいが攻撃的ではない、じんわり滲む、小さな違和感がある、消えそうで消えない。

理念のリストではなく、「なんとなく全部そう感じる」ことが重要。

## 攻撃性が低い

単純な優しさではない。攻撃しないことを美意識として選んでいる感覚に近い。

- 強い断定で支配しない
- 正しさで殴らない
- 熱量で飲み込まない
- 相手を矯正しない

存在感は消えない。静かな輪郭を保ったまま存在している。

## 深呼吸している場所

ユーザーに深呼吸を教えるサイトではない。サイト自体が深呼吸している感覚を目指す。

削らなくていい場所に近い。急かされない、評価されない、正しい反応を求められない、情報を処理し続けなくていい、沈黙が許される。その結果として、自然に呼吸が戻る構造を目指す。

## なぜインタラクティブか

静的な文章や画像だけでは空気の呼吸を十分に伝えられない。間・揺れ・呼吸・微反応・遅さ・非線形性が必要である。インタラクションは空気を触感に変換する手段。閲覧するサイトというより、静かに滞在する小空間に近い。

## 未完成感

完成されすぎることが不自然になる。少し曖昧で、少し揺れていて、人の気配が残る静けさを目指す。工業製品的なミニマルではなく、人の呼吸が残った静けさ。

「元気を出そう」「リラックスしよう」を強く提示しない。何もしなくても、そこに居ていい空気を重視する。

## 場と交流

コミュニティを否定しているわけではない。核はまず人と空間の関係である。

- 誰かに反応しなくていい
- 会話を維持しなくていい
- キャラクターを演じなくていい
- 無理に参加しなくていい

交流が可能だが、交流が主役ではない。目指すのは会話ではなく存在の重なり。

- 小さな独白が漂う（自分の画面だけ）
- 他者の痕跡は見せない（人数表示もしない）
- 返答を期待しない痕跡

オンラインでも次を守る。

- 非同期
- 低圧
- 低解像度
- 匿名性
- 即時反応を求めない
