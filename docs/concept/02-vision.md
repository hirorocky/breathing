# 場のビジョン

## 核

存在負荷が高い私たちが、同じ部屋で演じなくていい時間を試す場である。StackChan は人を導かず、正解を教えず、沈黙を埋めない。静かな輪郭を保ったまま机の上にいる。

## 空気の質感

静か、呼吸感がある、閉じていない、余白がある、説明しすぎない、思索的、やわらかい、寡黙、非攻撃的、未完成を許す、少し不器用、かわいいが幼くない、消えそうで消えない。

理念を個別に主張するのではなく、場全体から「なんとなくそう感じる」ことを目指す。

## 深呼吸している場所

参加者に深呼吸を教えたり、合わせるよう促したりしない。StackChan 自体が一定の呼吸を続け、人は視界の端にそのテンポがあるだけでよい。

急かされない、評価されない、正しい反応を求められない、情報を処理し続けなくていい、沈黙が許される。その結果として自然に呼吸が戻る場を探す。

## 身体が担うもの

- LCD とサーボによる共有の呼吸リズム
- 人間同士の視線を一時的に逃がせる第三の焦点
- LED、目、姿勢、短い音による弱い時間と内部状態の気配
- 固定周期ではない微細な揺らぎと、ゆっくりした回復

身体は人間の代わりにならない。同室の距離、体温、沈黙、言葉は人間同士が担う。

## 存在負荷の勾配

場は最初から低負荷とは限らない。

```text
高い ─────────────────────► 低い
入る・同席する             演じなくていられる
```

- 前半: 緊張や沈黙の重さを否定せず、言葉や操作を求めない
- 中盤: StackChan の呼吸と弱い気配が同じ時間を保つ
- 後半: 言葉や価値観が自然に出る余地を残す

この勾配を演出や手順で固定しない。実際に生まれたかを同席観察で確かめる。

## 美意識

- 強い断定、熱量、速さで支配しない
- かわいさで媚びず、キャラクターを演じすぎない
- 工業製品的な完全さより、息が残る不器用さを許す
- 存在感を消すのではなく、静かな輪郭を保つ
- 人の行動を評価し、報酬や罰を返す存在にしない
