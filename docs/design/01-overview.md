# アプリの全体像

## 何をするアプリか

ブラウザ上で呼吸する空間に滞在する。

## 体験

| 要素 | 振る舞い |
|---|---|
| 呼吸 | 中心の形が 8s 周期で揺らぐ |
| orb | 画面周辺に固定数の気配の点（最後が you） |
| イベント | 流れ星 / 風 / 波が稀に起きる |

利用者向け API は呼ばない。静的エクスポートしたフロントだけで完結する。

## 利用者の体験

**いる** — 何もしなくていい。

**見る** — 呼吸する形と、ゆっくりした気配を眺める。反応や評価は返ってこない。

## やらないこと

- トースト、バッジ、いいね、チャット、プッシュ通知
- 滞在時間・投稿数の表示、ランキング
- ユーザー登録、プロフィール、ソーシャルログイン
- 人数表示、言葉入力、他者の言葉の表示
- 退室アンケート、NPS、投稿直後の祝杯 UI
- 利用者向けアクセス解析

## 実装者向けデータ

利用者 UI には出さない。D1 と admin API（Worker）で観察する。

- おおよその利用状況（heartbeat / visit）
- API 利用回数（予算ガード用）
