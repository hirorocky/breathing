# アプリの全体像

## 何をするアプリか

ブラウザ上で呼吸する空間に滞在する。任意で短い言葉を置ける。オンライン有効時は、同時にいる人数だけが気配として見える。

## 動作モード

### オフライン（`NEXT_PUBLIC_ONLINE` 未設定）

API は呼ばない。従来どおりの単体体験。

| 要素 | 振る舞い |
|---|---|
| 人数 | 演出 fallback（3〜5人、`sessionSeed` 由来） |
| 言葉 | ブラウザ内のみ（最大 40） |
| orb | 固定数または演出 |

### オンライン（`NEXT_PUBLIC_ONLINE=1`）

| 要素 | 振る舞い |
|---|---|
| 人数 | API 成功時は実数。障害・予算超過時は fallback |
| 言葉（画面） | ローカルのみ |
| 言葉（サーバー） | POST で D1 保存（観察用。利用者向け GET なし） |
| orb | presence に連動（1〜7、最後が you） |
| 予算超過 | API 503。静的配信は継続 |

## 利用者の三つの体験

**いる** — 何もしなくていい。人数は「誰かいる」だけ。

**置く** — 任意キー → WordBar → Enter。24 文字、推敲不要、自分の画面にだけ漂う。

**見る** — 自分が置いた言葉が漂流し続ける。反応や評価は返ってこない。

## やらないこと

- トースト、バッジ、いいね、チャット、プッシュ通知
- 滞在時間・投稿数の表示、ランキング
- ユーザー登録、プロフィール、ソーシャルログイン
- 他者の言葉の表示
- 退室アンケート、NPS、投稿直後の祝杯 UI
- 利用者向けアクセス解析

## 実装者向けデータ

利用者 UI には出さない。D1 と admin API で観察する。

- 置かれた言葉の全文
- おおよその同時人数（heartbeat）
- API 利用回数（予算ガード用）
