## help と開示

help は許可の文章。アンケートや実験説明ではない。

**三層の開示**（先に許可を読ませ、最後に事実を置く）:

| 層 | 置き場所 |
|---|---|
| 常時 | 左下「この場について」→ `/privacy` |
| help | `?` 内の `help-disclosure` |
| 詳細 | `/privacy` |

**使う語:** 場をつくっている人 / 個人を特定しない / 何もしなくていい

**避ける語:** 監視、ログ収集、セッション ID、匿名化処理

**やらない開示:** 初回同意モーダル、投稿前後の注意ダイアログ、常時の「録画中」表示

### help-disclosure（確定文案）

```
この場は、存在の負荷を少し下げられるかを試す小さな試みです。
名前やプロフィールはありません。利用のおおよその様子だけを、個人を特定しない形で見ています。
```

### `/privacy` に書く項目

1. この場について（小さな実験）
2. 他の利用者に見えるもの（呼吸の形のみ）
3. 場をつくっている人に見えるもの（おおよその利用様子）
4. 目的（存在負荷を下げる設計の試行）
5. 第三者提供なし
6. 保管（古いものから削除）
7. 問い合わせ先
